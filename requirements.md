# Fluxa 需求文档

## 项目概述
Fluxa 是一个 AI 驱动的设计生成平台，用户可以通过自然语言对话创建视觉设计。平台采用 ops-driven 架构，所有画布修改都是离散、可重放的操作。

## 核心功能

### 1. 项目管理
- 用户可以创建和管理设计项目
- 每个项目包含多个文档（画布实例）
- 支持项目列表和缩略图显示

### 2. AI 对话设计
- 通过聊天界面与 AI 交互
- AI 根据用户提示生成画布操作
- 支持多种 AI 模型（GPT-4, Claude, Doubao）
- 实时反馈：立即响应 → 骨架 → 结果

### 3. 画布编辑器
- 基于 Fabric.js 的交互式画布
- 支持无限平移/缩放
- 图层选择和操作
- 导出 PNG 功能

### 4. 资产管理
- 用户上传参考图片
- AI 生成新图片
- 基于 Supabase 的安全存储

### 5. 认证和用户管理
- 邮箱/密码登录注册
- 邮箱验证
- 密码重置

## 用户故事

### 主要用户旅程
1. **发现和入门**
   - 用户访问主页，看到设计示例和快速开始选项
   - 注册/登录并验证邮箱

2. **项目创建**
   - 从主页创建新项目
   - 可选提供初始设计提示
   - 系统自动生成第一版设计

3. **迭代设计**
   - 在聊天中描述修改（"让文字更大", "添加日落背景"）
   - AI 生成相应画布操作
   - 更改立即在画布上显示
   - 用户可以手动调整元素或继续聊天

4. **资产管理**
   - 上传灵感参考图片
   - 根据描述 AI 生成新图片
   - 导出最终设计为 PNG 文件

5. **协作和分享**
   - 项目跨会话持久化
   - 实时更新支持多用户编辑（未来）
   - 通过导出图片分享设计

### 关键用户角色
- **内容创作者**：需要快速社交媒体图形、海报、营销材料
- **小企业主**：想要专业设计但没有设计技能
- **设计师**：使用 AI 快速原型和构思
- **营销人员**：通过对话创建活动视觉

## 技术架构
- 前端：Next.js 16 + React 19 + TypeScript + Tailwind CSS
- 后端：Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- 画布：Fabric.js
- UI 组件：Radix UI + shadcn/ui
- 状态管理：Zustand
- 实时：Supabase Realtime
- AI：OpenAI, Anthropic, VolcEngine 模型
- 测试：Vitest + 属性测试
